# TrengerJegParaply ☂️

TrengerJegParaply hjelper deg å finne ut om du bør ta med paraply før du setter deg på bussen, basert på været langs ruten din, akkurat nå.

## Funksjoner

- Sjekk om du trenger paraply før du tar bussen, Værvarsel langs din spesifikke bussrute
- Sanntidsdata for bussposisjoner og holdeplasser
- Basert på ruten din akkurat nå, ikke bare lokasjonen din
- Kun tilgjengelig i Norge

## Teknologi

- **React** + **Vite** — frontend-rammeverk og byggeverktøy
- **Tailwind CSS** — styling
- **shadcn/ui** — UI-komponenter
- **[Entur Journey Planner API](https://developer.entur.org/pages-journeyplanner-journeyplanner)** — sanntidsdata for bussposisjoner og holdeplasser
- **[Yr.no / MET Norway Nowcast API](https://api.met.no/weatherapi/nowcast/2.0/documentation#!/data/get_complete)** — værdata

## Kom i gang

```bash
npm install
npm run dev
```
