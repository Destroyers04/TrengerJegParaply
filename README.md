# TrengerJegParaply ☂️

TrengerJegParaply hjelper deg å finne ut om du bør ta med paraply før du setter deg på bussen, basert på været langs ruten din, akkurat nå.

## Features

- Check if you need an umbrella before catching the bus
- Weather forecast along your specific bus route
- Real-time bus positions and stop data
- Based on your route right now, not just your location

## Tech Stack

- **React** + **Vite** — frontend framework and build tool
- **Tailwind CSS** — styling
- **shadcn/ui** — UI components
- **[Entur Journey Planner API](https://developer.entur.org/pages-journeyplanner-journeyplanner)** — real-time bus positions and stop data
- **[Yr.no / MET Norway](https://developer.yr.no/)** — weather forecast data

## Getting Started

```bash
npm install
npm run dev
```
